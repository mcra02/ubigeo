## Uso

##### 1. Instalar requerimientos

```sh
pip install -r requirements.txt
```

##### 2. Generar archivos de migracion

```sh
python manage.py makemigrations
```

##### 3. Migracion de tablas

```sh
python manage.py migrate
```

##### 4. Iniciar servicio

```sh
python manage.py runserver
```
